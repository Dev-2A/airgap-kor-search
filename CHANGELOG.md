# Changelog

## [0.1.0] - 2026-02-19

### 추가

- 🔍 ONNX Runtime 기반 임베딩 모델 로더 (BGE-M3 기본, Mean Pooling 지원)
- 📄 문서 리더 (txt, md, pdf, docx) 및 한국어 텍스트 청킹
- 🗂️ FAISS 벡터 인덱스 + SQLite 메타데이터 저장소
- 🔎 시맨틱 검색 엔진 코어 (쿼리 → 임베딩 → 검색 파이프라인)
- 🖥️ CLI 인터페이스 (index, search, list, delete, stats, serve, init)
- 🌐 웹 UI (FastAPI + Vanilla JS, 다크 테마)
- 📚 모델 준비 가이드 및 에어갭 배포 가이드
